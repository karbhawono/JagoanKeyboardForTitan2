## Getting Started

### First-Time Setup

**Scenario:** You just installed Jagoan Keyboard and want to set it up.

1. **Enable the Keyboard**
    - Open Android **Settings**
    - Navigate to **System** ‚Üí **Languages & input** ‚Üí **Virtual keyboard**
    - Tap **Manage keyboards**
    - Enable **Jagoan Keyboard**

2. **Set as Default**
    - Open any text field (e.g., Messages app)
    - Tap the keyboard icon in navigation bar
    - Select **Jagoan Keyboard**

3. **Configure Basic Settings**
    - Open **Jagoan Keyboard Settings** from app drawer
    - Enable **Auto-Capitalize** (recommended)
    - Set **Preferred Currency** to your local currency (e.g., Rp for Indonesia)
    - Enable **Sticky Shift** if you want caps lock functionality
    - Choose **Suggestion Bar Mode** (recommend: AUTO)

## Vi Mode Navigation

### Example 11: Enabling Vi Mode

**Scenario:** You want to navigate text like a pro

**‚ö†Ô∏è IMPORTANT: Titan 2 Fn Key Setup Required**

Before using Vi mode, you must configure your Titan 2's Fn key:

1. Open **Titan 2 Keyboard Settings** (device settings, not keyboard app)
2. Navigate to **Fn Key Mapping** or **Function Key Settings**
3. Find **Fn key** option
4. **Map Fn ‚Üí TAB**
5. Save settings

**Why?** The Fn key on Titan 2 is not detectable by Android apps by default. Mapping it to TAB allows the keyboard to detect the **TAB+V+I** sequence.

**Steps to Enable Vi Mode:**

1. **Toggle Vi Mode:** Press **Fn+V+I** (which sends TAB+V+I)
2. **Verify:** Green bullet (‚óè) appears in suggestion bar
3. **Use Vi commands** (see examples below)
4. **Toggle off:** Press **Fn+V+I** again

**Note:** Throughout this guide, **TAB+V+I** means physically pressing **Fn+V+I** on your Titan 2 after mapping Fn to TAB.

### Example 20: Suggestion Bar Modes

**Scenario:** Choosing the right mode for your usage

**Mode 1: ALWAYS_SHOW**

- Shows whenever typing (with text/suggestions)
- Best for: Users who want constant feedback
- Example: Chat apps, emails

**Mode 2: AUTO**

- Shows when 2+ characters typed OR suggestions available
- Best for: Balanced approach (recommended)
- Example: General usage

**Mode 3: OFF**

- Never shows suggestion bar
- Best for: Minimal distraction, expert typists
- Example: Code editors, terminals

**Changing mode:**

1. Open **Jagoan Keyboard Settings**
2. Tap **Autocorrect**
3. Select **Suggestion Bar Mode**
4. Choose your preference

## Troubleshooting

### Autocorrect Not Working?

**Check:**

1. Settings ‚Üí Autocorrect ‚Üí **Enabled** ‚úì
2. Suggestion Bar Mode ‚Üí **AUTO** or **ALWAYS_SHOW**
3. Language ‚Üí **English** or **Indonesian** selected

### Symbols Not Appearing?

**Check:**

1. Long-press accents: Settings ‚Üí **Long-press Accents** ‚úì
2. Symbol picker: Tap **SYM** key
3. Category shortcuts: Hold **SYM** + letter key

### Vi Mode Not Activating?

**Check:**

1. Press **TAB+V+I** simultaneously
2. Look for green bullet (‚óè) in suggestion bar
3. Try toggling off and on again

### Number Formatting Issues?

**Check:**

1. Settings ‚Üí **Auto-Format Numbers** ‚úì
2. Must press **SPACE** after number
3. Field must not be password/email/URI type
